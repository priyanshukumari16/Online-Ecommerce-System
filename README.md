# Online-Ecommerce-System
The Online E-commerce Platform is a web-based system that connects Sellers, Buyers, and Administrators in a single marketplace.
It allows sellers to list products, buyers to purchase them, and admins to manage the overall system.
This project demonstrates full-stack web development using JSP, Servlets, JDBC, and MySQL.

#User Roles and Capabilities:
Admin	= Manage users, products, orders, system activity
Seller	= List products, manage inventory, process orders
Buyer	= Browse products, purchase items, track orders

##Core Features
 *Administrator:
Add / Edit / Delete Users
Approve and manage product listings
Track and update order status
Monitor system activity in real-time

 *Seller:
Add new product listings
Manage inventory availability
Process and ship customer orders
View sales performance reports and trends

* Buyer:
Browse and search products with filters
Add to cart and place orders
Track order status
Maintain wishlist and viewing history
Manage profile, address, and payment options

#Technology	Usage:
1.Java	= Backend Logic
2.JSP + Servlets	= Web UI & Controllers
3.JDBC	= Database Connectivity
4.MySQL	= Database
5.HTML / CSS / JSTL =	UI and templating
6.Apache Tomcat	= Deployment & Server
7.Maven	= Dependency Management

#How to Run the Project:
Import the project in Eclipse / NetBeans / IntelliJ as Maven Web Project
Configure MySQL database
Update DB credentials in DBConnection.java
Deploy the .war file on Tomcat Server
Open in browser
